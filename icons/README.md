# Icon Assets

This directory contains the extension icons.

## Required Sizes

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Source

The source SVG file is `icon.svg`. To generate PNG files:

1. Use an image editor (GIMP, Photoshop, etc.)
2. Open `icon.svg`
3. Export at the required sizes (16x16, 48x48, 128x128)
4. Save as PNG files with the corresponding names

## Online Tools

You can also use online SVG to PNG converters:
- https://convertio.co/svg-png/
- https://cloudconvert.com/svg-to-png

## Temporary Solution

For development, you can create simple colored PNG files using any image editor or use the SVG directly by converting it online.
